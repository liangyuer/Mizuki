# 计算思维基本知识点

**资料来源**：基于全套40份期末题库（涵盖约2800道真题）整理。 **复习策略**：这是最终完整版。请重点攻克 **Excel公式引用** 和 **网络协议/IP地址** 部分，这是最容易混淆的考点。

## 第一部分：Excel 电子表格 (⭐⭐⭐⭐⭐ 必考重灾区)

### 1. 单元格引用 (核心考点)

- **相对引用 (A1)**：公式复制到哪里，引用就变到哪里。
    
- **绝对引用 (**$A$**1)**：公式复制到任何地方，引用**都不变**。
    
- **混合引用 (**$A1 或 A$**1)**：
    
    - `$A1`：锁列（左右拖动列不变，上下拖动行变）。
        
    - `A$1`：锁行（上下拖动行不变，左右拖动列变）。
        
    - _口诀：$在谁前面就锁住谁。_
        
- **跨工作表引用**：`工作表名!单元格` (例如 `Sheet1!A1`)。
    

### 2. 数据输入技巧

- **文本型数字**：输入身份证号、学号等长数字，必须先输入英文单引号 `'` (如 `'2021001`)。
    
- **日期输入**：
    
    - 快捷键：`Ctrl + ;` (当前日期)。
        
    - 格式：必须用 `-` 或 `/` 分隔 (如 `2021-11-30`)。**用点号(2021.11.30)是错误的**，会被视为文本。
        
    - **分数输入**：输入 `0 1/2` (0+空格+分数)，否则变成日期。
        
- **自动填充**：
    
    - **数字**：直接拖动是“复制”，按住 `Ctrl` 拖动是“序列”(递增)。
        
    - **日期/文本+数字**：直接拖动默认是“序列”。
        

### 3. 常用函数与错误代码

- **必背函数**：
    
    - `SUM` (求和), `AVERAGE` (平均), `MAX/MIN` (最大/最小).
        
    - `COUNT` (只统计**数字**个数), `COUNTA` (统计非空单元格).
        
    - `RANK(数值, 范围, 0/1)`: 排名。
        
    - `IF(条件, 真值, 假值)`: 条件判断。
        
- **常见错误**：
    
    - `#####`：列宽不够 (拉宽列即可)。
        
    - `#DIV/0!`：除数为0。
        
    - `#VALUE!`：数据类型错误 (如 `1 + "A"`).
        
    - `#NAME?`：函数名拼写错误或引用了不存在的名称。
        

### 4. 数据分析操作

- **排序**：汉字默认按**拼音**排序，也可以按**笔画**排序。
    
- **分类汇总**：**操作前必须先对“分类字段”进行排序！**
    
- **筛选**：
    
    - **高级筛选**：同一行条件是“与”(并且)，不同行条件是“或”(或者)。
        
    - 条件区域的字段名必须与原表完全一致。
        
- **图表**：图表与数据源是**动态链接**的。修改数据，图表自动更新；删除数据，图表对应项消失。
    
- **极限值**：Excel 2010及以后版本，最大行号是 **1048576**，最大列标是 **XFD**。
    

## 第二部分：Word 文字处理 (⭐⭐⭐⭐ 易错细节)

### 1. 选定与编辑

- **选定技巧 (在左侧选定栏)**：
    
    - 单击：选定一行。
        
    - 双击：选定一段。
        
    - 三击：选定全文 (或 `Ctrl + A`)。
        
    - `Ctrl` + 单击句中：选定一句话。
        
    - `Alt` + 拖动：选定矩形块。
        
- **删除键**：
    
    - `Backspace` (退格键)：删除光标**前**的内容。
        
    - `Delete` (删除键)：删除光标**后**的内容。
        

### 2. 格式与排版

- **分节符 vs 分页符**：
    
    - **分页符 (Ctrl+Enter)**：仅换页。
        
    - **分节符**：**核心考点**。只有插入分节符，才能在同一文档中设置**不同的纸张方向** (一页横一页纵)、**不同的页眉页脚**、**不同的页边距**。
        
- **段落缩进**：
    
    - **首行缩进**：仅第一行缩进。
        
    - **悬挂缩进**：第一行不缩进，第二行及以后缩进。
        
- **换行符**：`Shift + Enter` (软回车，向下箭头)，换行但不分段。
    

### 3. 对象与邮件合并

- **图片环绕**：
    
    - **嵌入式** (默认)：图片像字符一样，不能随意拖动。
        
    - **四周型/紧密型/浮于文字上方**：可以随意拖动。
        
- **邮件合并**：用于批量打印工资条、信封。需要**主文档** (Word) 和 **数据源** (Excel/Access)。
    

## 第三部分：计算机网络与因特网 (⭐⭐⭐⭐ 理论重点)

### 1. IP地址与域名

- **IPv4**：32位 (4字节)，点分十进制 (如 192.168.1.1)。
    
- **IPv6**：128位，冒号十六进制，解决地址枯竭问题。
    
- **域名 (DNS)**：
    
    - DNS服务器负责将域名解析为IP地址。
        
    - 顶级域名：`.com`(商业), `.edu`(教育), `.gov`(政府), `.mil`(军事), `.cn`(中国), `.org`(非营利组织), `.int`(国际组织)。
        
- **子网掩码**：用于区分网络位和主机位。
    

### 2. 常见协议 (必背)

- **HTTP**: 超文本传输协议 (浏览网页)。
    
- **FTP**: 文件传输协议 (上传/下载)。
    
- **SMTP**: **发送**邮件。
    
- **POP3 / IMAP**: **接收**邮件。
    
- **TCP/IP**: 互联网最基本的协议。
    

### 3. 网络分类与硬件

- **覆盖范围**：LAN (局域网) < MAN (城域网) < WAN (广域网)。Internet是最大的WAN。
    
- **拓扑结构**：总线型、环型、**星型** (最常见，依赖中央节点，如交换机)。
    
- **硬件**：
    
    - **网卡 (NIC)**：物理地址 (MAC地址)，全球唯一，48位。
        
    - **路由器**：连接不同网络 (如局域网连入互联网)，选择路径。

## 第四部分：计算机基础与系统 (⭐⭐⭐ 概念题)

### 1. 计算机系统组成

- **冯·诺依曼体系**：核心思想是**存储程序**和**程序控制**。
    
- **硬件五大件**：
    
    - **CPU (中央处理器)** = **运算器** (ALU) + **控制器**。
        
    - **存储器**：`RAM` (断电丢失，速度快) vs `ROM` (断电不丢，存BIOS)。
        
    - **输入设备**：键盘、鼠标、扫描仪。
        
    - **输出设备**：显示器、打印机。
        
    - **双重设备**：磁盘驱动器、触摸屏、网卡。
        
- **性能指标**：**主频** (速度)、**字长** (精度/位数，如64位)。
    

### 2. 数据与进制

- **单位**：1 Byte (字节) = 8 bit (位)。
    
- **换算**：1 KB = 1024 B (注意是1024不是1000)。
    
- **进制**：
    
    - 二进制 (B)、八进制 (O)、十进制 (D)、十六进制 (H)。
        
    - 运算：二进制 `1+1=10` (逢二进一)。
        
- **编码**：
    
    - **ASCII**：`a` (97) > `A` (65) > `0` (48)。
        
    - **汉字编码**：一个汉字占2个字节 (GB2312)。
        

## 第五部分：PowerPoint 演示文稿 (⭐⭐⭐ 操作题)

### 1. 视图模式

- **普通视图**：编辑每张幻灯片。
    
- **幻灯片浏览视图**：**不可编辑内容**，专门用于**排序、复制、删除、设置切换效果**。
    
- **备注页视图**：添加演讲者备注 (放映时观众看不到)。
    

### 2. 动画与放映

- **切换** vs **动画**：切换是PPT页面之间的过场；动画是页面内文字/图片的动作。
    
- **SmartArt动画**：可以设置为“作为一个对象”播放，或“逐个”播放。
    
- **母版**：统一修改所有页面的Logo、背景、字体。
    
- **放映快捷键**：`F5` (从头播放), `Shift+F5` (从当前页播放), `Esc` (退出)。
    
- **隐藏幻灯片**：放映时不显示，但文件里并没有删除。
    

## 第六部分：Windows 操作系统 (⭐⭐ 基础操作)

### 1. 文件管理

- **文件名**：不区分大小写。非法字符：`\ / : * ? " < > |`。
    
- **通配符**：`?` 代表1个字符，`*` 代表任意个字符。
    
- **快捷方式**：扩展名 `.lnk`，删除它不影响源文件。
    
- **回收站**：
    
    - 硬盘删除 -> 进回收站 (可还原)。
        
    - **U盘删除** -> **不进**回收站 (直接彻底删除)。
        
    - `Shift + Delete` -> 直接彻底删除。
        

### 2. 常用快捷键

- `Alt + Tab`：切换窗口。
    
- `Alt + F4`：关闭窗口。
    
- `Ctrl + C/V/X`：复制/粘贴/剪切。
    
- `PrintScreen`：全屏截图到剪贴板。
    
- `Win + D`：显示桌面。
## ⚠️ 易错总结 (考前必看)

1. **关于“不”字题**：这是题库中最大的陷阱。例如“下列说法**不正确**的是”，考试时一定要看清。
    
2. **Excel的公式**：如果遇到 `$A$1` 这种绝对引用，直接选“复制后行列都不变”的选项。
    
3. **Word的分节**：如果题目问“如何让第一页竖向，第二页横向”，答案一定是“插入**分节符**”。
    
4. **U盘删除**：记住，U盘里删东西是**没有后悔药**的（不进回收站）。
    
5. **存储器**：只有 ROM (只读存储器) 和 硬盘/U盘 (外存) 断电数据不丢失，**RAM (内存) 断电一定丢失**。
    
6. **高级筛选**：条件区域的字段名必须和数据表完全一致，**不能自己随意写**。
    
7. **分类汇总**：必须**先排序**，否则汇总结果是错误的。
    
8. **IP地址**：每一段最大是255，超过255的都是非法IP (如 256.1.1.1 是错的)。
