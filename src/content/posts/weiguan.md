# 微观经济学期末核心考点总结（全面复习版）

这份总结基于您提供的课堂课件（2025级），涵盖了从导论到市场结构的全部核心内容。为了应对期末考试，特别增加了**计算公式速查**和**易错辨析**板块。

## 🧮 核心计算公式速查表（背诵）

| 知识点        | 核心公式                                                                    | 备注                       |
| ---------- | ----------------------------------------------------------------------- | ------------------------ |
| **均衡条件**   | $Q_d = Q_s$                                                             | 解方程求均衡价格 $P_e$ 和数量 $Q_e$ |
| **需求价格弹性** | $E_d = \frac{\Delta Q/Q}{\Delta P/P} = \frac{dQ}{dP} \cdot \frac{P}{Q}$ | 点弹性公式，注意取绝对值判断           |
| **交叉价格弹性** | $E_{xy} = \frac{\Delta Q_x/Q_x}{\Delta P_y/P_y}$                        | $>0$ 替代品；$<0$ 互补品        |
| **消费者均衡**  | $\frac{MU_x}{P_x} = \frac{MU_y}{P_y}$ 或 $MRS_{xy} = \frac{P_x}{P_y}$    | 效用最大化条件                  |
| **生产者均衡**  | $\frac{MP_L}{w} = \frac{MP_K}{r}$ 或 $MRTS_{LK} = \frac{w}{r}$           | 成本最小化/产量最大化条件            |
| **利润最大化**  | $MR = MC$                                                               | **万能公式**，所有市场结构通用        |
| **停止营业点**  | $P = AVC_{min}$                                                         | 低于此点关门                   |
| **完全竞争长期** | $P = MR = SMC = LMC = LAC = SAC$                                        | 利润为0，效率最高                |

## 第一章：导论 & 基本假设

### 1. 核心概念

- **稀缺性 (Scarcity)**：资源是有限的，而人的欲望是无限的。这是经济学产生的根源。
    
- **机会成本 (Opportunity Cost)**：为了得到某种东西而所要放弃的**另一些东西中的最大价值**。
    
    - _例子_：你逃课复习经济学的机会成本，是你原本上课能学到的知识，或者去打工能赚的钱（取最高者）。
        
- **理性人假设 (Rational Man)**：
    
    - **消费者**：追求效用（满足感）最大化。
        
    - **厂商**：追求利润最大化。
        
- **“看不见的手”**：指市场机制（价格机制），引导资源有效配置。
    

### 2. 实证 vs 规范

- **实证经济学 (Positive)**：研究“是什么”，不带价值判断。（例如：失业率上升了1%）
    
- **规范经济学 (Normative)**：研究“应该是什么”，带有主观价值判断。（例如：政府应该降低失业率）
    

## 第二章：需求、供给与均衡价格

### 1. 需求 (Demand) 与 供给 (Supply)

- **影响需求的因素**：商品本身价格（线上移动）、收入、相关商品价格（替代/互补）、消费者偏好、预期。（导致曲线平移）
    
- **影响供给的因素**：商品本身价格、生产成本（要素价格）、技术水平、政府政策。）。乏弹性）：提价能增加总收益（谷贱伤农）。**例子**：食盐、粮食。
    
- **税收负担**：税收总是更多地落在**缺乏弹性**（这就离不开这个商品）的一方身上。
    
    - 若需求缺乏弹性（主要由消费者承担）；若供给缺乏弹性（主要由生产者承担）。
        

## 第三章：消费者行为理论

### 1. 两种分析方法

- **基数效用论（边际效用分析）**：
    
    - **边际效用递减规律**：随着消费数量增加，每多吃一口带来的满足感是下降的。
        
    - **均衡条件**：$\frac{MU_x}{P_x} = \frac{MU_y}{P_y} = \lambda$（花在每种商品上的最后一单位货币带来的效用相等）。
        
- **序数效用论（无差异曲线分析）**：
    
    - **无差异曲线**：线上任意一点的效用相同。特征：凸向原点、互不相交、离原点越远效用越大。
        
    - **预算线**：$P_x \cdot X + P_y \cdot Y = I$。
        
    - **均衡点**：无差异曲线与预算线**相切**的点。
        
    - **公式**：$MRS_{xy} = \frac{P_x}{P_y}$（边际替代率 = 价格之比）。
        

### 2. 难点突破：替代效应与收入效应

- **正常品**：价格下降 $\rightarrow$ 替代效应增加，收入效应增加 $\rightarrow$ 总需求增加。
    
- **低档品**：价格下降 $\rightarrow$ 替代效应增加，收入效应减少 $\rightarrow$ 总需求增加（替代 > 收入）。
    
- **吉芬商品 (Giffen Goods)**：
    
    - **定义**：价格下降，需求量反而减少的特殊低档品。
        
    - **例子**：课件中提到的1845年爱尔兰大饥荒时的**土豆**。土豆价格上涨，人们穷得买不起肉，只能买更多土豆来充饥。
        
    - **特征**：收入效应（负向） > 替代效应（正向）。
        

## 第四章：生产理论

### 1. 短期生产（一种要素可变）

- **总产量(TP)、平均产量(AP)、边际产量(MP)** 关系：
    
    - $MP > AP$ 时，AP上升（像拉高平均分的学霸）。
        
    - $MP < AP$ 时，AP下降。
        
    - $MP = AP$ 时，AP达到最大值。
        
- **边际报酬递减规律**：在技术不变和其他要素固定时，连续增加一种可变要素，边际产量最终会下降。
    
    - **例子**：三个和尚没水喝。一个厨房里厨师太多，反而互相碍事，效率降低。
        
- **生产三阶段**：厂商理性的生产区域是**第二阶段**（AP最大点到MP=0之间）。
    

### 2. 长期生产（所有要素可变）

- **最优要素组合**：等产量曲线与等成本线相切。
    
- **公式**：$\frac{MP_L}{w} = \frac{MP_K}{r}$ 或 $MRTS_{LK} = \frac{w}{r}$。
    
- **规模报酬**：
    
    - 投入增加1倍，产出增加 > 1倍 $\rightarrow$ 规模报酬递增。
        
    - 投入增加1倍，产出增加 < 1倍 $\rightarrow$ 规模报酬递减。
        

## 第五章：成本理论

### 1. 成本概念

- **机会成本**：做一件事必须放弃的其他事情中价值最大的那个。
    
    - **例子**：课件中提到的小王下海经商，他的机会成本不仅是投入的资金，还有他**辞职放弃的工资**和**资金存银行的利息**。
        
- **经济利润** = 总收益 - 经济成本（显性+隐性）。
    
    - 注：经济利润为0不代表没钱赚，只代表获得了**正常利润**。
        

### 2. 成本曲线（必考图形）

- **短期**：
    
    - MC（边际成本）曲线呈U型。
        
    - MC穿过AC（平均成本）和AVC（平均可变成本）的**最低点**。
        
- **长期**：
    
    - LAC（长期平均成本）是无数条SAC曲线的包络线。
        
    - **规模经济**导致LAC下降，**规模不经济**导致LAC上升（U型）。
        

## 第六章：完全竞争市场（重点）

### 1. 市场特征

- 无数买者卖者、产品同质（完全一样）、进出自由、信息完全。
    
- 厂商是**价格接受者**（Price Taker），需求曲线是一条水平线。
    
- 公式：$P = AR = MR$。
    

### 2. 短期均衡

- **均衡条件**：$MR = SMC$ （即 $P = SMC$）。
    
- **盈亏分析**：
    
    - $P > SAC$：盈利。
        
    - $P = SAC$：盈亏平衡。
        
    - $AVC < P < SAC$：亏损但继续生产（能弥补全部可变成本和部分固定成本）。
        
    - $P \le AVC$：**停止营业点**（关门大吉）。
        

### 3. 长期均衡

- 厂商利润为零（只能获得正常利润）。
    
- **条件**：$MR = LMC = SMC = LAC = SAC$。
    

## 第七章：不完全竞争市场

### 1. 垄断市场 (Monopoly)

- **特征**：唯一的卖者，产品无替代品，进入壁垒极高。
    
- **均衡**：$MR = MC$。但注意，此时 $P > MR$，也就是价格高于边际成本。
    
- **价格歧视**：
    
    - 一级：按消费者愿意支付的最高价卖（榨取所有消费者剩余）。
        
    - 二级：按购买数量分组定价（如批发价、阶梯电价）。
        
    - 三级：对不同市场的不同人群定价（如学生票、成人票）。**条件**：市场的弹性不同。
        

### 2. 垄断竞争 (Monopolistic Competition)

- **特征**：类似牙膏、洗发水市场。厂商很多，但产品**有差别**。
    
- **长期均衡**：
    
    - 利润为零，但**存在多余的生产能力**（没有在LAC最低点生产）。
        
    - 图形特点：需求曲线d与LAC相切，但不是切在最低点。
        

### 3. 寡头市场 (Oligopoly)

- **特征**：几家大厂商控制市场（如通信运营商、石油）。
    
- **模型**：
    
    - 古诺模型（产量竞争）。
        
    - 斯威齐模型（弯折的需求曲线）：解释价格刚性（跟跌不跟涨）。
        

## ⚠️ 期末易错点与避坑指南

### 1. 需求变动 vs. 需求量变动

- **陷阱**：把“价格变化”引起的需求变化叫“需求变动”。
    
- **正解**：
    
    - **需求量变动**：只有**商品本身价格**变化引起的，表现为**点在同一条线上移动**。
        
    - **需求变动**：由收入、偏好、其他商品价格等非价格因素引起的，表现为**整条曲线的平移**。
        

### 2. 停止营业点 vs. 盈亏平衡点

- **陷阱**：认为亏损（$P < AC$）就应该马上关门。
    
- **正解**：
    
    - **盈亏平衡点**：$P = AC$。此时经济利润为0。
        
    - **停止营业点**：$P = AVC$。只要 $P > AVC$，虽然亏损，但每卖一件还能赚回一点钱来弥补固定成本（如房租），所以**短期内**应继续营业。只有连可变成本（如人工、材料）都赚不回来时，才关门。
        

### 3. 边际报酬递减 vs. 规模报酬

- **陷阱**：混淆“短期”和“长期”的生产规律。
    
- **正解**：
    
    - **边际报酬递减**：是**短期**规律，前提是**技术不变，其他要素固定**（如机器不变，只增加工人）。
        
    - **规模报酬**：是**长期**规律，前提是**所有要素同比例**增加（如工人和机器都增加一倍）。
        

### 4. 经济利润 vs. 会计利润

- **陷阱**：算出经济利润为0，觉得这个生意白做了。
    
- **正解**：
    
    - 经济利润 = 总收益 - (显性成本 + 隐性成本)。
        
    - **经济利润为0**意味着你获得了**正常利润**，即你的回报和把资源投到其他地方（机会成本）是一样好的。
        

### 5. 垄断厂商没有供给曲线

- **陷阱**：试图画出垄断厂商的供给曲线。
    
- **正解**：完全竞争厂商的短期供给曲线是 $SMC$ 曲线大于 $AVC$ 的部分。但在垄断市场中，同样的价格可能对应不同的产量，同样的产量可能对应不同的价格（取决于需求曲线的移动），因此**不存在**规律性的供给曲线。
